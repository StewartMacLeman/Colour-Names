# Colour Names App.

An app that displays the various ways to represent a colour when using CSS.

## Build with:
* Express
* Mongoose
* EJS
* JavaScript
* CSS
---
## Seed data.

Add the documents used with the below terminal command.

```
npm run seeds
```
